<div className="subheading">Getting started</div>

Get started by installing **WCS** in your project :
```
npm install wcs-core
yarn add wcs-core
```

We highly recommend to use the **[Template guidelines](../?path=/docs/documentation-template--documentation)**.  
If you are using a specific framework, please refer to **[Framework integrations](../?path=/docs/documentation-framework-integrations--documentation)**.  

<div className="subheading">Contribute</div>
You can contribute by adding new issues for bugs or new features on the official **[WCS GitLab repository](https://gitlab.com/SNCF/wcs)**.  
Feel free to submit a merge request that will be reviewed by our team.
